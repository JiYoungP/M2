# M2
Milestone 2 for CS-2340
Person 2 is responsible for taking in a string, and outputting that String's characters as a new string, but in 
random order. 

METHODS:

public Person2(String pname) 
	Constructor, this takes in the person's name

private String calc(String input) 
	Modifies the input string by randomizing the characters in input
	RETURNS: a string with randomized characters

public String toString(String input) 
	Outputs a string representation of the Person2 object.
	RETURNS: a string which contains the data in this Person2 object.
